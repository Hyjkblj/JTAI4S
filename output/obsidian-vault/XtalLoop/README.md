# XtalLoop Obsidian Demo Vault

这个 Vault 用于演示 XtalLoop 的个人知识复用端。

## 使用方式

1. 用 Obsidian 打开本目录。
2. 若 Obsidian 提示安全模式，请仅在本地演示环境中启用社区插件。
3. 打开 XtalLoop Importer 插件，或直接查看下面的入口文件。

## 入口

- [[XtalLoop/Index|XtalLoop Index]]
- [[XtalLoop/Graph/Relationships|Relationships]]
- [[XtalLoop/Meetings/MTG-DEMO-002|Meeting MTG-DEMO-002]]
- [[XtalLoop/Experiments/EXP-DEMO-002|Experiment EXP-DEMO-002]]

## 边界

- 本 Vault 只保存脱敏 demo bundle。
- 飞书仍是组织事实源。
- Obsidian 只做个人阅读、复盘、双链和图谱浏览。
